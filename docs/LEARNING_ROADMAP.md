# Learning Roadmap

This roadmap keeps the project useful as a portfolio artifact while staying inside a clearly authorized local lab.

## Milestone 1: Local Weaknesses

- Plaintext preference storage.
- Hardcoded static secret.
- Sensitive debug logging.
- Mutable local entitlement state.

## Milestone 2: Safer Comparisons

- Keychain-backed credential storage.
- Redacted logging helper.
- Server-authoritative entitlement model.
- Clear separation between local cache and authorization.

## Milestone 3: iOS Simulator Target

- Add an Xcode project or generated project file.
- Run the same labs on iOS Simulator.
- Add screenshots for each lab state.
- Document simulator-specific storage inspection paths.

## Milestone 4: Instrumentation Practice

- Add local-only observation exercises for LLDB.
- Add Frida notes only for this lab binary and simulator build.
- Compare source-level findings with runtime observations.

## Milestone 5: Portfolio Polish

- Add screenshots and a short demo GIF.
- Keep generated study reports in a private notes folder or a sanitized sample.
- Add GitHub topics such as `swift`, `swiftui`, `ios-security`, and `security-lab`.
